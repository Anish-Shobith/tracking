# Title
tracking
# Description
The backend API used to track the documents
## Tech Stack: 
- Express
- Prisma
- Mongodb

# How to run the hack ?

Clone the project:
```bash
git clone git@github.com:Bytenamics/tracking.git
```
Change Directory:
```bash
cd tracking
```
Install dependencies : 
```bash
yarn install
```
Run:
```bash
yarn dev
```
